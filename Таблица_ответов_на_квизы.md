## 2. Linear regression: bias, variance, MSE


| Вопрос                                        | Правильно ✓                                             | Неправильно                                          |
| --------------------------------------------- | ------------------------------------------------------- | ---------------------------------------------------- |
| Linear regression — какая модель?             | **High bias, low variance**                             | Low bias, только high variance                       |
| MSE = V(f̂) + Bias²(f̂) + Vε: bias и variance | **Увеличение Bias² → уменьшение V(f̂)** (trade-off)     | Bias и variance «двигаются одинаково»                |
| Связь с overfitting / underfitting            | **V(f̂) связана с overfitting; Bias² — с underfitting** | Наоборот — нет                                       |
| Train MSE vs test MSE                         | **Train MSE обычно ниже test MSE**                      | Test MSE всегда выше — «всегда» слишком жёстко       |
| Переобучение (overfitting)                    | **Даёт низкую train MSE**                               | —                                                    |
| Больше предикторов в линейной регрессии       | **Может уменьшить ошибку на train**                     | Не «всегда» lower training MSE — бывают контрпримеры |

## 10. Bias–variance (MSE, eq. 2.7)

Формула: E[y₀ − f̂(x₀)]² = V(f̂) + Bias²(f̂) + Vε


| Правильно ✓                                     |
| ----------------------------------------------- |
| V(f̂) связана с **overfitting**                 |
| Bias²(f̂) связана с **underfitting**            |
| Увеличение Bias² → уменьшение V(f̂) (trade-off) |
| V и Bias² **не** двигаются в одну сторону       |

## Краткая шпаргалка: что заучить наизусть

```
Y = Response (target)
ML ⊂ AI,  SL ⊂ Statistics
Linear regression → high bias, low variance
Train MSE обычно < test MSE
Overfitting → низкая train MSE
Logistic: MLE, log-likelihood (не LSE!)
Log odds линейна по X и β₁; Odds — нет
Balance лучше Income для default
P[D|B,R] может быть <0 или >1; logistic — в [0,1]
F-test: все β=0? Отклонение → хотя бы один ≠0
√R² = Corr(X,Y); предпочитаем меньший RSE
Больше предикторов: может уменьшить train error (не «всегда»)
Bias² ↑ → Variance ↓; Variance ~ overfitting, Bias² ~ underfitting
```
